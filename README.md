# Dr. Ashish & Dr. Ayushi — Wedding Invitation

A premium, interactive digital wedding invitation replicate.

## Features
- **Cinematic Entry**: immersive video gate on entry.
- **Floating Petal Animation**: Dynamic HTML5 Canvas animation.
- **Music Integration**: Instrumental background score with custom controls.
- **Countdown Timer**: Real-time tracking to the big day.
- **Responsive Design**: Optimized for all devices with premium typography and shimmer effects.
- **RSVP System**: Interactive form with digital memory book integration.

## Local Setup
1. Clone this repository.
2. Open `index.html` in any modern web browser.

## Assets
All media assets (videos, images, audio) are stored in the `/assets` directory for complete offline compatibility.
